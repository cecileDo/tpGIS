## installation

Pour installer et creer l'environement permettant d'executer les scripts du tp:

```shell
sudo apt install libcairo2-dev pkg-config python3-dev
git clone https://github.com/cecileDo/tpGIS.git
cd tpGIS
make venv
make dependencies
```
